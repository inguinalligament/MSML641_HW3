# Comparative Analysis of RNN Architectures for Sentiment Classification

# Setup Instructions:
## Dependencies
Python 3.11  
TensorFlow 2.20.0  
scikit-learn 1.2+  
pandas 2.1+  
matplotlib 3.8+

You can install all dependencies using:
pip install -r requirements.txt

## How to run:
### python train.py --data ./data/IMDB_Dataset.csv --epochs 6 --batch 32 --results ./results/metrics.csv



# Expected runtime and output files
### 60 minutes CPU, 30 minutes GPU
### metrics.csv
